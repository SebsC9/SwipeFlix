# SwipeFlix

Aplicacion web tipo "Tinder" pero con peliculas, muestra peliculas aleatorias al usuario, el usuario tendra la opcion de agregarla a la watchlist o seguir con otra pelicula.
Desarrollado con **REACT + VITE + TAILWINDS**, Usando la API de **TMDB**
