# SwipeFlix

Aplicacion web tipo "Tinder" pero con peliculas, muestra peliculas aleatorias al usuario, el usuario tendra la opcion de agregarla a la watchlist o seguir con otra pelicula. Tambien tiene pagina de search para que el usuario pueda buscar lo que quiere
Desarrollado con **REACT + VITE + TAILWIND**, Usando la API de **TMDB**
